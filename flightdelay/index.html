✈️ Flight Delay Prediction

Tools used: Python, Scikit-learn, XGBoost, Pandas, Tableau

This project explores over 100 million flight records to build predictive models for estimating delay duration. The goal is to identify major causes of delays and help airlines improve scheduling and operations.

🔍 Project Highlights

  • Built and compared multiple models: Linear Regression, Random Forest, XGBoost
  • Used extensive feature engineering (weather, airport, carrier info)
  • Evaluated using RMSE, MAE, and R² scores
  • Presented results in a Tableau dashboard
